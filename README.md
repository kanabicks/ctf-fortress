# 🏰 CTF Fortress

**CTF Fortress** — это ультимативный автоматизированный комплекс для разведки и эксплуатации, разработанный специально для соревнований **Attack-Defense CTF**, пентестинга и аудита инфраструктуры.

Инструмент объединяет скорость Bash-скриптов с мощью промышленных сканеров (Nuclei, SQLMap, Ffuf, Katana, Hadolint), выступая в роли "швейцарского ножа" для кибербезопасности.

## 🚀 Возможности

### 🔴 Red Team (Режим Атаки)
- **Умная Разведка:** Определение технологического стека, анализ WAF, быстрое сканирование портов (Nmap).
- **Глубокий Краулинг:** Рекурсивный сбор ссылок через Katana и фаззинг директорий через Ffuf.
- **Авто-Эксплуатация:**
  - **SQL Injection:** Автоматическое обнаружение и эксплуатация через SQLMap.
  - **XSS / RCE:** Анализ параметров с помощью Dalfox и Commix.
  - **Поиск Уязвимостей:** Проверка по 5000+ шаблонам Nuclei (CVE, мисконфигурации).
- **Облака и API:** Поиск открытых S3 бакетов, интроспекция GraphQL и Swagger документации.
- **Режим Наставника:** В конце сканирования генерирует "Пошаговую инструкцию по эксплуатации" для ручного добивания цели.

### 🔵 Blue Team (Аудит Инфраструктуры)
- **Аудит Dockerfile:** Проверка на запуск от `root`, наличие секретов в ENV, опасные инструкции (интеграция с Hadolint).
- **Аудит Docker Compose:** Обнаружение `privileged: true`, проброса `docker.sock` и отсутствия лимитов ресурсов.

### 📊 Отчетность
- **Markdown Отчет:** Подробный файл `FINAL_REPORT.md` с критическими находками.
- **Визуальная Карта:** Генерация карты атаки в формате Mermaid (`.mmd`) для наглядного представления векторов.

## 🛠️ Установка

1. **Клонируйте репозиторий:**
git clone https://github.com/kanabicks/ctf-fortress.git
cd ctf-fortress


2. **Сделайте скрипт исполняемым:**
chmod +x fortress.sh


3. **Установите зависимости (один раз):**
Запустите скрипт и выберите пункт **4**.
sudo ./fortress.sh
Выберите: 4) Setup / Update Tools



---

## 🕹️ Использование

Запускайте скрипт с правами root для полной функциональности (Nmap требует привилегий):

sudo ./fortress.sh


### Опции Меню:
1. **Web Attack (A/D CTF)** – Полный цикл авто-взлома целевого URL.
2. **Infrastructure Audit (Dockerfile)** – Статический анализ файлов сборки Docker.
3. **Infrastructure Audit (Compose)** – Аудит безопасности конфигураций запуска.
4. **Setup** – Установка необходимых инструментов (Go, Python, Wordlists).
